# PNGToBin

PNGファイル（Jpegなどもいけると思います）をARGB8888のバイナリに変換します。  
画像をそのままFPGAなどのメモリに展開するのにお使いください。  

## 使い方
ビルドしたプログラム（もしくはビルド済みのもの）に画像をドラッグアンドドロップしてください。  
画像が保管されていた場所にファイル名.binで変換したものを保存します。
